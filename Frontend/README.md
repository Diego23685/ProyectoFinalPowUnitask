# login-cra

App creada con Create React App (estructura mínima) con un formulario de login **sin lógica de autenticación**.

## Cómo usar
```bash
npx create-react-app login-cra
cd login-cra
# Reemplaza los archivos generados por los de este zip (src/ y public/ y package.json)
npm install
npm start
```
